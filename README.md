# Tibia Compendium
Tibia Compendium is a [Tibia](http://www.tibia.com/abouttibia/?subtopic=whatistibia) Android app. This app is developed mainly for personal use and as a learning excercise.

## Current features
* Character lookup
* Guild lookup
* Item lookup

## Planned features

* Level up notifications
* Damage calculators
* Stat calculators
* Creature lookup
